# C.O.D.E. Atlas Reviewer Onboarding

Reviewer onboarding acknowledgements are stored in `user_onboarding_acknowledgements`.

For each assigned Pilot project and schema, the reviewer must acknowledge the schema instructions version and hash. Pilot annotation submission is blocked until the acknowledgement exists. If the schema instructions hash changes, the reviewer must acknowledge the new version.

Guidelines are available at `/guidelines/<schema_id>` and can be opened without losing a Review Workspace draft.
